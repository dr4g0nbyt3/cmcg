# Changelog

All notable changes to this project will be documented in this file.

## [0.1.0] - 2026-04-08

### Added
- Initial `.cmcg` manifest schema (TypeScript interfaces)
- Core `CMCGPlayer` — video + canvas overlay with `requestAnimationFrame` render loop
- `SourceResolver` — priority chain: variable, local, remote, fallback
- Sample manifest with two image slots (banner + logo)
- Animated gradient base video for demo
- SVG placeholder fallback assets
- Product design document (v0.3) with market research and AI integration strategy
